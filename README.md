# Netflix Content Analysis Dashboard (Power BI)

## 📌 Project Overview
This project delivers a comprehensive data analysis of Netflix's content library using Power BI. The goal is to uncover deep insights into the streaming giant's media strategy, analyzing content shifts over time, geographic concentration, and genre distributions.

## 🛠️ Data Source & Structure
The dataset used for this analysis is `netflix_titles.csv`, which includes metadata for movies and TV shows available on Netflix.
* **Data Folder:** Contains the raw CSV file.
* **Assets Folder:** Contains the visual elements used in the report and documentation.

## ⚡ Key Features & Insights Uncovered
* **Content Strategy:** Tracked the rapid growth of Movies vs. TV Shows, identifying the peak production years.
* **Geographical Insights:** Visualized the top content-producing countries, highlighting market domination.
* **Rating Distributions:** Analyzed the target audience demographics (e.g., TV-MA, TV-14) across different media types.

## 🚀 Tech Stack Used
* **Power BI Desktop:** For report authoring and dashboard visualization.
* **Power Query:** Used for data cleaning, handling missing values, and changing data types.
* **DAX (Data Analysis Expressions):** Created custom measures for advanced metrics.
